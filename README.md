# Stopword Remover

Remove stopwords from text

## Features

- Zero external dependencies (stdlib only)
- Easy-to-use CLI interface
- Professional Python implementation
- MIT licensed

## Installation

```bash
pip install -e .
```

Or clone and install:

```bash
git clone https://github.com/Viprasol-Tech/stopword-remover
cd stopword-remover
pip install -e .
```

## Usage

### Python

```python
from stopword_remover import StopwordRemover

result = StopwordRemover.process("data")
print(result)
```

### CLI

```bash
python -m stopword_remover "your input here"
```
